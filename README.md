# Freelancing_BD 🇧🇩

A desktop-based **Freelancing Platform** built with **C# (Windows Forms / WPF)** and **SQL Server**, developed as an academic project. The platform connects freelancers with clients, allowing them to post jobs, place bids, and manage work orders.

---

## 📌 Features

- 🔐 User registration and login (Client & Freelancer roles)
- 📋 Job posting by clients
- 💼 Bid/proposal submission by freelancers
- 📊 Dashboard for managing projects and orders
- 💳 Payment and transaction tracking
- 🔍 Search and filter jobs by category

---

## 🛠️ Technologies Used

| Technology     | Purpose                  |
|----------------|--------------------------|
| C#             | Application logic (OOP)  |
| Windows Forms  | User Interface (UI)      |
| SQL Server     | Database management      |
| T-SQL          | Stored procedures/queries|
| ADO.NET        | Database connectivity    |

---

## 🗄️ Database

The project uses **SQL Server** with relational tables including:
- `Users` — stores client and freelancer accounts
- `Jobs` — job listings posted by clients
- `Bids` — proposals submitted by freelancers
- `Orders` — confirmed work between client and freelancer
- `Payments` — transaction records

---

## 🚀 How to Run

1. Clone the repository: `git clone https://github.com/talimulutsha/Freelancing_BD.git`
2. Open the solution file (`.sln`) in **Visual Studio**.
3. Open **SQL Server Management Studio (SSMS)** and run the SQL script to create the database.
4. Update the **connection string** in the project to match your SQL Server instance.
5. Build and run the project from Visual Studio.
